# Ahmedabad Yatra 🚇🚌

> **A Smart Multi-Modal Public Transportation Platform for Ahmedabad**

Ahmedabad Yatra is an ongoing software engineering project that aims to simplify public transportation in Ahmedabad by integrating the city's metro and bus networks into a single intelligent platform. The long-term vision is to provide users with efficient route planning, travel time estimation, fare calculation, and real-time transit information through an intuitive mobile and web application.

The project is currently in the **Database Design Phase**, where the complete database architecture has been designed and implemented as the foundation for future application development.

---

## 📌 Project Status

**Current Phase:** Database Design Completed | Application Development In Progress

### Completed

* Database design for the Ahmedabad Metro System.
* Database design for the Ahmedabad Bus System.
* Entity Relationship (ER) Diagram.
* Relational Schema.
* SQL DDL scripts.
* SQL Data Insertion scripts.

### In Progress

* Backend development.
* Route planning algorithms.
* Metro and bus integration.
* Application architecture.

## 🎯 Project Objectives

The primary objective of Ahmedabad Yatra is to develop an intelligent transportation platform that enables commuters to:

* Search optimal routes between any two locations.
* Integrate metro and bus transportation into a single journey.
* View estimated travel time and fare.
* Reduce unnecessary route changes.
* Improve daily commuting efficiency.
* Provide scalable infrastructure for future real-time services.
